# proyecto-pdi

Remontemos.


Proyecto del curso PDI

(El contenido de esta descripción se modificará a lo largo del curso)

Proyecto a implementar:

-Abrir Imágenes .BMP mediante el método Scanline

-Convertir del modelo RGB a algún otro modelo de color

-Convertir de color a grises utilizando el rango medio

-Visualizar el histograma de una imagen (canales R,G,B,Intensidad) en un mismo componenete TChart

-Binarización (intensidad, R,G,B) solicitando umbral mediante Trackbar y visualizando el promedio de la intensidad

-Filtro gamma solicitando el respectivo parámetro

-Ajuste automático de contraste

-Suavizado recortado con parámetros (a,t) a=extremos a descartar, considerando máscaras de t x t

-Algún filtro para detección de bordes mostrando éstos en color negro y las otras regiones en blanco

-Textura codificada considerando regiones de 3x3

-Aumento de escala (factor 2xy) de manera interpolada

-Reducción de escala (factor 1/2 xy) de manera interpolada

-Rotación a la izquierda y a la derecha

-ALU para imágenes considerando las operaciones y especificaciones del diseño/interfaz mostrada en clase

-Mostrar espectro de Forier de una imagen (en escala de gris)

*Funcionalidades adicionales:

-Restaurar imagen

-Guardar como...

****NOTAS:**

-El proyecto debe implementarse en Lazarus 2.0.12 siguiendo los paradigmas, estructuras y ejemplos de códigos fuente desarrollados en clase.

-El proyecto se desarrolla de manera individual.

-Cualquier actitud no ética (plagio, falsedad, códigos desarrollados por I.A.) se penalizan con la respectiva invalidación de la evaluación.
